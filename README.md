# MVVMsimple

A simple iOS calculator app demonstrating MVVM architecture with RxSwift.

## Overview
This small app performs basic arithmetic operations (add, subtract, multiply, divide).  
It uses RxSwift to bind ViewModel outputs to the View.

## Tech
- Swift
- UIKit
- RxSwift
- MVVM

## Features
- Add, subtract, multiply, divide two numbers
- Reactive UI update using RxSwift

## Usage
Open the project in Xcode, install pods (if any), and run on simulator or device.
